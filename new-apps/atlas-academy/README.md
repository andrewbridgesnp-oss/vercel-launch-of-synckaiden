# Atlas Academy 2026 – Interactive Trainer (MVP)

Offline-first React app that helps you master AI primitives and ship automations daily.

## What’s in here
- Core modules + lessons + streaks + notes (persisted to localStorage)
- Daily plan generator
- Intel Pack library (JSON imports)
- Workflow blueprint builder (Markdown output)
- 25 AI monetization paths list
- Ethical social autopilot starter

## Run
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## What Copilot should report (high-level)
- Single-page app: `src/App.jsx`
- UI primitives: `src/components/ui/*`
- Offline-first: localStorage only
- No secrets, no network calls
